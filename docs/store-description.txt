FlixMonkey adds IMDb ratings directly to Netflix thumbnails, with Metacritic and Rotten Tomatoes available from supported providers, so you can spot better picks faster.

Ratings appear on title cards, hover previews, detail views, and search results. Available as a Chrome extension, Firefox add-on, and Tampermonkey/Violentmonkey/Greasemonkey userscript.

⭐ FEATURES
• View IMDb ratings directly on Netflix titles, with Metacritic and Rotten Tomatoes when available
• Click the IMDb badge to open the title on IMDb, or search IMDb when no match is found
• Skip low-rated titles by fading out anything below your chosen IMDb threshold

⚙️ CUSTOMIZATION
• Position rating badges in any corner of thumbnails
• Show or hide Metacritic and Rotten Tomatoes scores
• Set the IMDb rating threshold for the fade-out feature

📊 RATING PROVIDERS
FlixMonkey supports multiple data sources:
• FM-DB + Agregarr (default): IMDb ratings with no API key or setup needed
• OMDb: Recommended for full IMDb, Metacritic, and Rotten Tomatoes coverage with a free API key
• XMDb: Free API key for IMDb and Metacritic
• IMDb API (deprecated): Unreliable due to low rate limits

⚡ CACHING
Ratings are cached locally for instant loading:
• Older titles (1+ year) are cached indefinitely
• Recent titles refresh monthly to capture updated ratings
• Unrated titles are retried after 24 hours
• Failing providers are temporarily paused to prevent repeated errors
